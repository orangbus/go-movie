# 仿YouTube在线聚合影视
> 前提条件，已经安装了docker以及docker-compose

## 如何使用

## 1、下载源码
```
git clone https://github.com/orangbus/go-movie.git
```

## 2、运行
```bash
docker-compose up -d
```
访问：http://ip:3000
